# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-05-26

### Added
- **Multi-currency Support**: Support for NGN, USD, and USDT wallets.
- **Ledger-based Accounting**: Immutable transaction logging for all debit and credit movements.
- **Real-time mock exchange rates**: Interactive currency conversion between supported currencies with custom transaction fee calculations.
- **Send & Receive Money**: P2P transfers via Virtual Account Number or email address; dynamic QR Code payment request generator.
- **Profile & Settings**: Custom screens for transaction PIN management, biometric security settings, and app themes.
- **Admin Dashboard**: Manage and verify users, view global transaction activity, and process manual withdrawal requests.
- **Test Automation**: Laravel feature test suite for auth/wallet APIs, Flutter widget tests, and Newman/Postman automated API tests.
